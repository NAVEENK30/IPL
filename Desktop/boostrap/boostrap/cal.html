Soil Moisture Detector

The Soil Moisture Detection System uses an Arduino, GSM module, and MIT App Inventor to provide real-time monitoring of soil moisture levels.

Into The Green

This project aim to develop a Soil Moisture Dedection System using Arduino a soil moisture secnsor & a GSM Module communication with a custom -built Mobile Application created using MIT APP INVENTOR.The system measures soil moiture levels and transmits the data to the mobile application,ensuring effective irrigation management.

ARDUINO UNO

The Arduino microcontroller processes data from the soil moisture sensor and controls other components, such as the relay and GSM module. It enables automated irrigation based on soil conditions and allows for easy programming and adjustments.

RELAY MODULE

The relay module acts as a switch that controls the water pump. When the soil moisture level drops below a set threshold, the Arduino activates the relay to turn on the pump, automating the irrigation process.

BLUETOOTH MODULE

The HC-05 Bluetooth module is a popular, affordable, and widely-used Bluetooth v2.0 + EDR module. It operates on a 3.3V or 5V power supply and has a range of up to 100 meters. The module can be used for wireless communication between two devices, such as connecting a microcontroller to a smartphone or computer.

SOIL SENSOR

The soil moisture sensor measures the volumetric water content in the soil by detecting electrical conductivity. It provides real-time data to determine when irrigation is needed, helping to conserve water and enhance plant health.

GSM MODULE

A GSM module or a GPRS module is a chip or circuit that is used to establish communication between a mobile device or a computing machine and a GSM or GPRS system.

WATER PUMP

A water pump motor is an electric or diesel-powered motor used to pump water in various applications, such as irrigation systems, water treatment plants, and residential water supply systems. These motors are designed to provide a reliable and efficient means of moving water, often in high-pressure and high-volume applications.

Code

#include /SoftwareSerial.h/
const int sensorPin = A0;
const int pumpPin = 7;
const int bluetoothTx = 10;
const int bluetoothRx = 11;

SoftwareSerial bluetooth(bluetoothTx, bluetoothRx);
void setup() {
Serial.begin(9600);

pinMode(pumpPin, OUTPUT);
digitalWrite(pumpPin, HIGH);

Serial.println("System Initialized");
bluetooth.println("Bluetooth Connected"); }
void loop() {

int sensorValue = analogRead(sensorPin);

float moisturePercentage = (100.0 - (sensorValue / 10.23));

Serial.print("Moisture Percentage: ");
Serial.print(moisturePercentage);
Serial.println("%");

bluetooth.print("Moisture Percentage: ");
bluetooth.print(moisturePercentage);
bluetooth.println("%");

if (bluetooth.available()) {
String command = bluetooth.readStringUntil('\n');
if (command == "PUMP_ON") {
digitalWrite(pumpPin, LOW);
bluetooth.println("Pump: ON");
Serial.println("Pump: ON");
} else if (command == "PUMP_OFF") {
digitalWrite(pumpPin, HIGH);
bluetooth.println("Pump: OFF");
Serial.println("Pump: OFF");
} else {
bluetooth.println("Invalid Command");
}
}
delay(2000);
}
